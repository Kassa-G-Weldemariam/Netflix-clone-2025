# Netflix Clone 2025

A responsive frontend clone of the Netflix landing page built to practice modern UI layout and JavaScript interactivity.

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript (React)

## 🎯 Features
- Responsive navigation bar
- Hero section with featured content
- Movie thumbnail grid layout
- Hover effects and interactive UI components
- Mobile-friendly responsive design

## 📚 What I Learned
- Structuring complex layouts using Flexbox and Grid
- Writing clean, modular JavaScript
- Improving UI/UX consistency
- Responsive design principles

## ▶️ How to Run
1. Clone the repository
2. Open `index.html` in your browser

## 📌 Purpose
This project was built for frontend practice and to strengthen my understanding of real-world UI structure.
